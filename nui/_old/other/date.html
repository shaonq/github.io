前言

现在的大多数项目需要使用到日历js，比如用来实现签到，快速选择某一天等类似的目的。在这基础上开发一个分离css和js的日历插件。

API和功能演示
demo 

    //依赖：jQuery
    //加载插件
    date(document.getElementById("f_date"), {
       // url:"/date/json", 要预先加载的数据
        data:function(a,b){
            if(b)for(var i in b){var d1 = a.join("-"),d2 = b[i];
                if(d1===d2){
                    $(this).css({"color":"blue"}).find("p").text("签到")
                }
            }
         }
        ,click: function (d) {
            alert("日期：" + (d.join('-')))
        }
    })
    //下面的表格样式是我的默认表格样式

api 
初始化插件：data(this,options)
     其中 this：是指要显示这个表格的js选择器(非jQ), 或者$("#f_date")[0](jQ)
          options：选项 见下表格
 header:表头字段 {left:"", right:""}
 date:当前时间
 url：ajax提交URL
 data：数据处理fn(a,b) a表格数据，b ajax返回数据
 click： fn表格数据 a 当前日期

    //ajax 数据格式
 {
     "status":0,
     "data":[//下面数据可以自定义
          "2016-06-01",
          "2016-07-01",
          "2016-08-01",
          "2016-09-01",
          "2016-10-01",
          "2016-11-01",
          "2016-12-01"
     ]
 }
header 是否需要表头，为true就显示月分，同时可传入 {left:"", right:""}
date 日历初始化的哪一天
url ajax请求数据的URL
data 处理url请求成功的数据
click function:回传参数为数组([年，月，日])
