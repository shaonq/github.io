nui.cookie
/**
    按f12，Application -> Storage -> Cookies ,可查看当前域名下的cookie
    nui.cookie(key) //读取一个cookie
    nui.cookie(key,value)  //设置一个带session性质的cookie
    nui.cookie(key,value,options) //设置一个cookie
*/
    nui.log("","nui.cookie demo","green")
    nui.cookie("cookie",'{"a":"aaa","b":"bbb"}')
    nui.log(nui.cookie("cookie"),"cookie")
属性 描述
options.expires 生效天数
options.path 生效范围
options.domain 有效域名
options.secure 是否https可读

nui.util
属性 描述
*format yyyy-MM-dd HH:mm:ss / yyyy-MM-dd / yyyy-MM
nui.util.randomSize(min,max,fixed) 生效min-max之间的随机数,fixed为小数位数
nui.util.date.now(format) 获取当前时间
nui.util.date.format(new Date()，format) 格式化一个有效时间
nui.util.date.toDate('2017-01-01') 将日期装换为new Date()对象
nui.util.date.countDown('2018-01-01') 倒计时 
nui.log("","nui.util.date demo","green")
nui.log(nui.util.date.now(+1,"yyyy-MM-dd HH:mm:ss"),"Tomorrow Date")
nui.log(nui.util.date.toDate("2015-01-01"),"2015-01-01 toDate")
nui.log(nui.tpl("还有{{d.d}}天{{d.H}}小时{{d.m}}分{{d.s}}秒就到2019年了",nui.util.date.countDown('2019-01-01 00:00:00')),"友情提示","red")
